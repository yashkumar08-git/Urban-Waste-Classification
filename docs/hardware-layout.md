# Hardware Layout Design

## Objective
To define the physical layout of sensors and motors in the smart waste bin.

## Layout Description
- IR sensor placed near the lid to detect waste insertion.
- Servo motor attached to the lid for automatic opening and closing.
- Rain sensor positioned inside the bin to detect wet or dry waste.
- Stepper motor placed below the bin to rotate the waste compartment.

## Result
Hardware layout finalized for stable system performance.
