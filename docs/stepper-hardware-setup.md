# Stepper Motor Hardware Setup

## Objective
To verify proper connection and rotation of the stepper motor using the ULN2003 driver.

## Components
- Stepper Motor (28BYJ-48)
- ULN2003 Driver Module
- Arduino Uno

## Connections
IN1 → Pin 8  
IN2 → Pin 9  
IN3 → Pin 10  
IN4 → Pin 11  
VCC → 5V  
GND → GND  

## Observations
- Motor rotates clockwise and counterclockwise.
- Rotation speed stable.
- No overheating observed.

## Result
Stepper motor hardware setup completed successfully.
