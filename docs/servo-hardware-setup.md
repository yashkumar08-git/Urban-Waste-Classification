# Servo Motor Hardware Setup

## Objective
To configure and test the servo motor used for automatic lid control in the smart waste bin.

## Connections
- Red Wire → 5V
- Brown/Black Wire → GND
- Yellow/Orange Wire → Arduino Pin 9

## Observations
- Servo rotates correctly.
- No voltage fluctuation observed.
- Power supply remains stable.

## Result
Servo motor hardware setup completed successfully.
