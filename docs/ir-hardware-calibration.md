# IR Sensor Hardware Calibration

## Objective
To calibrate the IR sensor for reliable object detection.

## Connections
- VCC → 5V
- GND → GND
- OUT → Arduino Pin 2

## Observations
- Detection range approximately 5–10 cm.
- Onboard LED indicates object detection.
- Sensor sensitivity adjusted using potentiometer.

## Result
IR sensor successfully calibrated for object detection.
