# Servo Motor Testing 

## Objective
To test basic servo motor rotation using Arduino Uno.

## Connection Details
- Red wire → 5V
- Brown/Black wire → GND
- Yellow/Orange wire → Pin 9

## Working
The servo rotates to:
- 0 degrees
- 90 degrees
- 180 degrees

Each position holds for 2 seconds.

## Result
Servo motor successfully tested and working correctly.
