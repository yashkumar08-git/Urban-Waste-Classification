# System Architecture 

## Objective
To design the overall architecture of the Urban Waste Classification system.

## System Components
- Arduino Uno
- Servo Motor (lid control)
- IR Sensor (object detection)
- Rain Sensor (wet/dry detection)
- Stepper Motor (waste redirection)

## System Workflow
1. IR sensor detects object.
2. Servo motor opens bin lid.
3. Rain sensor identifies wet or dry waste.
4. Stepper motor directs waste to appropriate compartment.

## Result
System architecture finalized for hardware integration.
