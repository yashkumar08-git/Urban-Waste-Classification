# Final Hardware Assembly

## Objective
To assemble all hardware components of the Urban Waste Classification system into a single working setup.

## Components Assembled
- Arduino Uno
- Servo Motor
- IR Sensor
- Rain Sensor
- Stepper Motor
- Stepper Motor Driver

## Assembly Steps
1. Mounted servo motor on the bin lid.
2. Positioned IR sensor at the bin entry point.
3. Installed rain sensor inside the detection area.
4. Attached stepper motor to the waste compartment mechanism.

## Result
All hardware components successfully assembled for system operation.
