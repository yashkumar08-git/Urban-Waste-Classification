# Rain Sensor Calibration

## Objective
To calibrate the rain sensor for detecting wet and dry conditions.

## Connections
- VCC → 5V
- GND → GND
- AO → Arduino A0

## Observations
Dry Sensor Value: ~800  
Wet Sensor Value: ~350

## Threshold Value
Threshold set at approximately 500.

## Result
Rain sensor successfully distinguishes between wet and dry conditions.
