# System Workflow

## Objective
To explain the working process of the intelligent waste classification system.

## Workflow
1. User approaches the smart waste bin.
2. IR sensor detects waste insertion.
3. Servo motor opens the bin lid.
4. Rain sensor checks the moisture level.
5. The system determines whether the waste is wet or dry.
6. Stepper motor rotates the compartment to the correct section.

## Result
The system automatically detects and classifies waste.
