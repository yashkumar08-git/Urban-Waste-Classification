# Hardware Planning 

## Objective
To review and confirm the hardware components required for the Urban Waste Classification IoT system.

## Components Verified
- Arduino Uno
- Servo Motor
- IR Sensor
- Rain Sensor
- Stepper Motor
- Stepper Motor Driver
- Jumper Wires
- Breadboard

## Work Done
- Verified availability of all required hardware components.
- Ensured compatibility with Arduino Uno.
- Planned wiring layout for sensors and motors.

## Result
All required hardware components are available and ready for implementation.
